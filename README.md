# HR-Attrition-Analysis-Dashboard

## Project Overview
This project focuses on analyzing employee attrition within a company using an HR dataset. The goal is to identify patterns, understand employee demographics, and uncover the key drivers of attrition to support better workforce management.
The analysis was conducted in Excel with the data cleaned and transformed in Power Query (loaded as connection only). Visualizations and dashboards were created to present insights clearly.

# Note: The raw HR dataset is included in this repository. The cleaned and transformed data used in the dashboard exists only in Power Query (connection-only) and is not included in this repository.
